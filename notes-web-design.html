HTML structures content; CSS styles it. We look at semantic tags, responsive layouts and color theory HTML (HyperText Markup Language) It is the main language used to create web pages. It provides the structure or framework of a website. It tells the browser what to display ‚Äî such as text, images, links, videos, and headings. Everything you see on a webpage (words, pictures, buttons) is placed there using HTML. Think of HTML as the skeleton or foundation of a webpage. CSS cascading sheets It is used to style and design the webpage created with HTML. CSS controls how things look ‚Äî like colors, backgrounds, fonts, sizes, spacing, and layouts. It makes websites beautiful, organized, and user-friendly. Think of CSS as the clothing and decoration that make the HTML skeleton look good. It can also make web pages responsive (fit well on phones, tablets, and computers).
